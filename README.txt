Inspire Me - Simple Quote Management App

Description:
-------------
Inspire Me is a basic web application for generating, saving, editing, and deleting quotes. It consists of a front-end interface for displaying and manipulating quotes from both an external API and a local database. The app allows you to fetch random quotes, save them to your database, edit existing quotes, and delete quotes you no longer need.

Features:
---------
1. Generate and display random quotes from an external API.
2. Save your favorite quotes to a local database.
3. Edit existing quotes with customized content and author.
4. Delete quotes from your saved collection.

Instructions:
-------------
1. Click "New Quote" to fetch and display a random API generated quote.
2. Click "Save Quote" to save the displayed API quote to your local database.
3. Click "Edit" on any saved quote to modify its content or author.
4. Click "Delete" to remove a quote from your database.

Requirements:
--------------
- RapidAPI Key
- RapidAPI Subscirption to "Quotes" By Martin Svoboda 
- Web server (e.g., Apache) to host the application.
- PHP installed for server-side processing.
- A MySQL database for storing quotes.

Installation:
--------------
1. Clone or download the 'Inspire Me' project files to your web server directory.
2. Create a MySQL database named 'quotes_database' (or your preferred name).
3. Import the 'quotes' table into your database using command-line tools.
4. adjust the database connection settings in 'quote_db_connection.php' and 'update.php' accordingly to your set up.
6. Open your web browser and access the application.
